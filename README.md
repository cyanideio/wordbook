# backbone-7
Backbone.js under Framework7 \w Coffeescript, Gulp, Browserify, Sass

## Getting Started

####Install global gulp if not yet present.

`npm install -g gulp`

####Install and run

`npm install`

`gulp fetchStatic`

`npm start`
